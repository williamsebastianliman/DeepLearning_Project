HOW TO RUN THE APPLICATION

RUNNING THE BACKEND
1. Go to Root Folder
2. Install All Dependencies In requirements.txt
3. Run python -m uvicorn app.backend.main:app

RUNNING THE FRONTEND
1. Go to app/frontend
2. Run npm run dev

Used Model In the Application is named quickdraw_scratch_final_deploy.keras
Model is trained on notebooks/modelling.ipynb
